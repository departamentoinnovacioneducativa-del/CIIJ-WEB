Unidad 1: Estadística descriptiva y exponentes

REACTIVO 01
1. Durante el primer parcial de Matemáticas, Ana obtuvo las siguientes calificaciones en 5 exámenes: 8.5, 9.0, 7.5, 8.0 y 9.5. ¿Cuál es el promedio final?

Suma de muestras dividida entre n=5:

Suma = 8.5 + 9.0 + 7.5 + 8.0 + 9.5 = 42.5 Promedio = 42.5 / 5 = 8.5
REACTIVO 02
2. El número de libros que leyeron 7 estudiantes de primer semestre durante las vacaciones fue: 3, 1, 4, 2, 5, 1, 6. ¿Cuál es la mediana de libros leídos?

Se ordenan los datos: 1, 1, 2, 3, 4, 5, 6. Como n=7 (impar), el valor central en la posición (7+1)/2 = 4 es el resultado.

Frecuencias absolutas: Metro (8 repeticiones), Camión (4), Bicicleta (2), Auto (2). La categoría con mayor frecuencia es el Metro.

REACTIVO 15
15. Modelado lineal: c) Una granja tiene cerdos y gallinas, en total hay 35 cabezas y 116 patas. ¿Cuántos cerdos y gallinas hay?

Ecuaciones: c + g = 35 y 4c + 2g = 116. Multiplicando la primera por -2:

-2c - 2g = -70 4c + 2g = 116 -------------- 2c = 46 -> c = 23 cerdos. Sustituyendo: g = 35 - 23 = 12 gallinas.
REACTIVO 16
16. Resuelve por Fórmula General la ecuación cuadrática: 2x² - 7x + 3 = 0.

Fórmula: x = [-b ± √(b² - 4ac)] / 2a. a=2, b=-7, c=3:

x = [7 ± √((-7)² - 4(2)(3))] / 4 x = [7 ± √(49 - 24)] / 4 = [7 ± 5] / 4 x₁ = 12/4 = 3 || x₂ = 2/4 = 0.5
